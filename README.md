# contDiteAds

Sistema de controle de clientes e funcionários — registra o que cada funcionário faz para cada cliente (tarefas, horas, status, observações).

## Stack
A definir após inspeção do ambiente Hostinger.

## Perfis
- **Admin**: vê todos os clientes, todos os funcionários e todas as atividades.
- **Funcionário**: vê apenas os clientes/atividades atribuídos a ele.
